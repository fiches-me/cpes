# Analyses néoclassiques et keynésiennes de l'arbitrage consommation/épargne
## L'Analyse néoclassique intertemporelle
En 1930, Fisher publie un livre, <u class="red-underline"><strong>Théorie de l'intérêt</strong></u>, qui réfléchis sur le taux d'intérêt, qui symbolise un peut le prix du temps. Il y a l'idée d'épargner/des-épargner.

Comme tout les néoclassiques, il part de S= I. Il n'y a donc aucun excès d'épargne, **aucune sur-épargne** dans le modèle de Fisher. Une forme d'épargne est improductive : la thésaurisation. Or, les néoclassiques exclue ce cas de leurs analyses car Say. Ils considère donc une épargne "équilibré", elle trouve toujours un investissement correspondant. Il va donc chercher à comprendre les **préférences individuelles d'épargne**. Les consommateurs sont majoritairement contraint par le temps, il n'est pas possible de consommer trop aujourd'hui car il faut l'argent nécessaire.  Il propose donc une équation, appelé équation de Fisher, :
$$ i = r + \pi^e$$
Où $i$ est le taux d'intérêt <u class="wrong-underline">nominal</u>, $r$ est le taux d'intérêt <u class="wrong-underline">réel</u> et $\pi^e$ l'inflation espérée (**anticipée**). La contrainte temporelle est ici approché d'un point de vue **monétaire**.  Il considère également que les consommateurs préfèrent **consommer aujourd'hui**. La consommation donne plus d'utilité dans le présent que dans le futur.
