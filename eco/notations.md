# Notations
## Notations Générales

- $W$ : salaires, (origine : Wages)
- $P$ : prix
- $\frac{W}{P}$ : salaire réel
- $L$ = travail (origine: Labor)
- $C$ = Consommation
- $\gamma$ = Production
- $S$ = Épargne (origine: X)
- $I$ = Investissement
- $R$ = Real interest rate
