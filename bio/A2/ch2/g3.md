## Coordination de l'activité des cellules

Les nutriments, le gaz, certaines molécules doivent êtres échangés entre cellules. *Il y  a une nécessité de communication entre les cellules, mais comment ?* 

> [!DÉFINITION]
> La communication cellulaire est tout processus qui assure le transfert **d'informations** *uniquement* entre deux cellules d'un même organisme 

### Communication **juxtacrine**

Communication entre **deux cellules adjacentes**. Deux possibilités :
1. [Jonctions Gap](g2.md#Tableau%20récapitulatif%20des%20Jonctions)
2. Protéines membranaire de reconnaissance (voir [Anticorps](../../A1/ch1/g5.md#Anticorps%20Reconnaissance%20et%20Défense) pour un exemple)

### Communication **paracrine**

Communication **à courte distance** (*du même tissus*). C'est une communication par **émission d'un facteur diffusif** qui va donc agir à proximité de la cellule qui livère ce facteur. 

### Communication **synaptique**

Autre communication **à court terme** qui agit sur les cellules environnantes.  Un neurone secrète des molécules d'un neurotransmetteur dans la fenêtre synaptique. 
Polarise-Dépolarise. Il faut dépasser un certain seuil de simulation pour créer un message électrique. 

### Communication à distance (**hormonale**)

Utilisation d'hormones pour transmettre des messages
Exemple : [Glucagon et Insuline](../../A4/ch3/g3.md#Mécanismes%20de%20régulation)

> [!WARNING]
> Une communication hormonale n'est pas forcément à distance (c.f. [Communication paracrine](#Communication%20**paracrine**))

### Communication **électrique**

Autre communication dans les neurones, mais ici dans l'*axone*. Entre 2 cellules nerveuses (neuro-neuronales) ou entre un neurone et une autre cellule (neuro-musculaire, neuro-*x*). Le flux électrique permet alors, ou non, une action dans la cellule.
Centre nerveu -> neurone -> cellule
Ce qui sort de la cellule nerveuse est éléctrique. Le lien est paracrine, mais l'information est électrique.
