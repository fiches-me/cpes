## 3D Game

```python
class Player:
    def __init__(self, x, y, z):
        self.x = x
        self.y = y
        self.z = z

    def coord(self): return (self.x, self.y, self.z)

    def front(self): self.x += 1 
    def back(self):  self.x -= 1 
    def left(self):  self.y += 1
    def right(self): self.y -= 1
    def up(self):    self.z += 1
    def down(self):  self.z -= 1
```
