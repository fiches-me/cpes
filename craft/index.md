# CPES-CRAFT

![title](title.png)

Vous l'auriez deviné, **CPES-CRAFT** est le serveur Minecraft des CPES. Vous trouverez ici les informations utiles pour vous connecter et bien démarrer.

## Connexion
Le serveur est accessible en version **1.21.11** via l'adresse `cpes.funa.dev`. Il est accessible aux comptes officiels et crack.
