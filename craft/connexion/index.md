# Rejoindre le serveur

## Étapes rapides

1. Installe un launcher Minecraft (voir [Installer Minecraft](./installation.md)).
2. Lance la version **1.21.11**.
3. Ajoutez le serveur : `cpes.funa.dev`.
4. Connecte-toi.

## Vous avez un compte officiel

Rien de particulier à faire : connectez-vous avec votre compte Microsoft dans votre launcher, puis rejoignez le serveur.

## Vous jouez en crack

Ça fonctionne aussi, avec quelques précautions. Le guide est ici : [Comptes Crack](./crack.md).

## Vérifications avant de demander de l'aide

- Vérifiez la version du jeu : **1.21.11**.
- Vérifiez l'adresse : `cpes.funa.dev`.
- Évitez TLauncher (explications dans [Installer Minecraft](./installation.md)).
- En cas de besoin de mods/optimisation : [Mods (facultatifs)](./mods.md).
- Si une vérification est activée : [Vérification](./verification.md).
